## Features
- Clean and modern design
- Responsive layout for desktop and mobile devices
- Built with Astro and Tailwind CSS for optimal performance and maintainability
- Easy to navigate with clear calls to action

## Technologies Used
- Astro: A modern static site generator that allows for fast and efficient website development.
- Tailwind CSS: A utility-first CSS framework that provides a wide range of pre-designed classes for building responsive and visually appealing websites.
- Alpine.js: A lightweight JavaScript framework used for adding interactivity to the website without the need for a full JavaScript framework.
