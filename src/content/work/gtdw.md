## Features
- Clean and modern design
- Responsive layout for desktop and mobile devices
- Built with Astro and classic CSS for optimal performance and maintainability
- Easy to navigate with clear calls to action

## Technologies Used
- Astro: A modern static site generator that allows for fast and efficient website development.
- Classic CSS: Traditional CSS used for styling the website, providing a solid foundation for design and layout without the need for additional frameworks.
- No JavaScript frameworks were used, keeping the website lightweight and focused on content.
